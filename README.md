# HR Employee Attrition Analysis

## Objective
To analyze employee attrition and identify factors affecting employee turnover.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Dataset
The dataset contains employee information such as department, salary level, age, years at company and attrition status.

## Analysis Performed
1. Data Cleaning
2. Attrition Analysis
3. Department-wise Analysis
4. Salary Analysis
5. Experience Analysis

## Conclusion
Employees with low salary and less experience tend to leave the company more frequently.
